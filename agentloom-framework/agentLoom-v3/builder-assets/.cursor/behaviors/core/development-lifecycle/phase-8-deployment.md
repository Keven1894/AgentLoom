# Phase 8: Documentation & Deployment

**Parent**: [AgentLoom Development Lifecycle](./INDEX.md)

---

## Goal

Finalize user-facing documentation and release the agent.

---

## 8.1 Complete Documentation

- **README.md**: The project landing page.
- **User Guides**: Step-by-step instructions for end-users.
- **Troubleshooting**: Solutions for common problems.

## 8.2 Create Onboarding

- **Start Guide**: `NEW_AGENT_START_HERE.md` or similar.
- **Quick Reference**: Cheat sheets for commands/skills.
- **Examples**: Sample workflows or prompts.

## 8.3 Deploy & Monitor

- **Git**: Commit and push all changes.
- **Release**: Tag the version (e.g., v1.0.0).
- **Monitor**: Plan for feedback collection and future updates.

---

**Output**: A released, documented agent ready for users.
