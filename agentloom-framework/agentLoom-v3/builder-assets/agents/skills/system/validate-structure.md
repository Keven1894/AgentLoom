# Validate System Structure

## Description
A capability to check if the file system matches the expected architectural layout.

## Usage
Run the validation script to ensure all required directories and core files are present.

## Execution
```bash
python scripts/validate_structure.py
```

## Prerequisites
- Python environment
- `scripts/` directory must exist
