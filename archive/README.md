# archive/

Historical AgentLoom material kept for provenance only. **Nothing here is part of
the runnable framework** — the live, installable package lives in `src/agentloom/`.

| Path | What it is |
|------|------------|
| `protocol/agentLoom-v3/` | The v3 protocol package: 9-phase setup manuals, design docs, and legacy builder-assets (superseded by `src/agentloom/`). |
| `protocol/dual-helix-3tracks-startup-v1..v3/` | Earlier dual-helix startup prototypes. |

These files are retained so the design lineage of the framework is auditable. For
current usage, see the top-level [`README.md`](../README.md).
